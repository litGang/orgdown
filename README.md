# orgdown notebook
this repository is a notebook project build with electron, react, redux. and data store in nedb.


a desktop client for orgdown editor.

![](./screenshot.png)


# How to build?
1. git clone https://github.com/ic4907/orgdown.git
2. cd orgdown
3. npm install
4. npm install electron -g
5. npm run start
6. electron .
